#  CoffeeHouse – Coffee Shop Website

##  Project Overview
**CoffeeHouse** is a responsive and interactive web application designed for a modern coffee shop.  
The website showcases the coffee shop’s brand, menu, customer testimonials, gallery, and contact information using a clean and user-friendly interface.

This project was developed as a **Final Project for the Web Programming course (2025)**.

---

##  Project Objectives
- Create a professional online presence for a coffee shop  
- Apply core web programming concepts (HTML, CSS, JavaScript)  
- Build a responsive and visually appealing website  
- Practice teamwork, planning, and frontend development

##  Features
- Responsive navigation bar with hamburger menu  
- Hero section with call-to-action buttons  
- About section with social media links  
- Menu section displaying coffee and food categories  
- Testimonials slider using **Swiper.js**  
- Image gallery  
- Contact section with contact details and contact form  
- Interactive UI elements (hover effects, icons, animations)  
- Cross-browser and responsive design  

---

##  Technologies Used
- **HTML5** – Structure  
- **CSS3** – Styling and responsive design  
- **JavaScript (Vanilla JS)** – Interactivity  
- **Swiper.js** – Testimonials slider  
- **Font Awesome** – Icons  
- **GitHub** – Version control
